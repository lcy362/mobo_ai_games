# 姓氏挑战大赛 - 完整规则手册

## 一、游戏目标
在10个维度上，根据姓氏限制说出历史人物或虚拟人物，获得尽可能高的总分，成为擂主并保持擂主地位。

> **每轮给出姓氏限制时，必须附带该维度的具体说明/描述。**

## 二、维度说明

| 维度 | 说明 | 满分代表人物（示例） |
|------|------|-------------------|
| 领军能力 | 统率军队、带领团队作战或竞争的能力 | 韩信 |
| 治理能力 | 治理国家、管理组织或团队行政体系 | 诸葛亮 |
| 个人武力 | 个人武力值、武艺高强、格斗能力、现代体育运动竞技能力 | 项羽 / 李小龙 |
| 谋略智慧 | 谋略、计策、推理、智商、运筹帷幄 | 张良 / 福尔摩斯 |
| 人情世故 | 外交斡旋、人际交往、洞察人性、驭人之术 | 晏婴 / 韦小宝 |
| 实务才能 | 经济规划、组织运营、制度建设、工程统筹 | 管仲 |
| 人文造诣 | 文化贡献、思想深度、文学造诣 | 孔子 |
| 个人魅力 | 感召力、领导魅力、容貌魅力 | 周公旦 / 貂蝉 |
| 科技贡献 | 科学技术、医学、工程技术、发明创造 | 张衡 |
| 后世影响 | 对后世、社会、文化、行业的影响力与地位 | 秦始皇 |

> 注：以上满分代表仅为示例，非唯一答案。虚拟人物可突破10分上限至12分。

### 姓氏选择核心规则
- **类别和姓氏都必须随机**：先随机决定用大/中/小哪一类，再从对应池中随机取姓
- **禁止指向性**：不得根据维度内容选择姓氏
- **严格按数量**：大姓只给1个，中等给2-3个，小姓给4-6个

## 四、评分标准详解

### 历史人物评分（0-10分）

| 分数 | 标准 | 要求 |
|------|------|------|
| 10 | 历史巅峰 | 该领域公认的TOP级别 |
| 9 | 顶尖 | 该领域超一流人物 |
| 8 | 卓越 | 该领域一流人物 |
| 7 | 优秀 | 有重要成就和影响力 |
| 6 | 良好 | 有显著成就但存短板 |
| 5 | 中等 | 有一定建树但不够突出 |
| 4 | 及格 | 有所作为但成就有限 |
| 3 | 偏弱 | 表现一般 |
| 2 | 差 | 表现较差 |
| 1 | 很差 | 该领域表现差或无作为 |
| 0 | 无相关 | 无相关事迹 |

### 虚拟人物评分（0-12分）

| 分数 | 标准 | 要求 |
|------|------|------|
| 11-12 | 超越历史 | 作品广为人知，能力明显强于所有真实人物 |
| 0-10 | 同历史标准 | 作品知名度不够或未明显超越 |

## 五、擂主规则
1. 第一轮胜者自动成为擂主
2. 后续每一轮与擂主打擂
3. 总分超过擂主则成为新擂主
4. 擂主记录各维度答案+总分
5. 每轮结果展示时必须逐维度对比擂主

## 六、人物不可重复
- **同一轮内已答人物不可再用**：不同维度不可给出相同人物
- **不得与擂主重复**：答案不得与当前擂主在**同一维度**的答案相同
- **跨轮次可重复**（但挑战擂主建议少重复）
- 违反上述规则需重新作答
